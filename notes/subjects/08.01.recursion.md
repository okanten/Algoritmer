## Base case

The base case is the case of which the call is not recursive any more (i.e the exit condition). The base case is there to terminate the recursion.

If there is no base case then you stumble upon a problem called _infinite recursion_, which is almost identical to an infinte loop, the only difference being that the "loop" occurs in the definition (defined method) itself. It is therefore critical to make sure to always include a base case.

> Any recursive definition must have a nonrecursive part, called the base case, that permits the recursion to eventually end.


_Defined by John Lewis & Joseph Chase, in Java Software Structures: Designing and Using Data Structures_

## Recursion in math

Factorial (n!) is an example of recursion in math. 

> In mathematics, the factorial of a positive integer n, denoted by n!, is the product of all positive integers less than or equal to n:

```
n! = n * (n - 1) * (n - 2) * (n - 3) ... * 3 * 2 * 1
```


```
5! = 5 * 4 * 3 * 2 * 1 = 120
```

## Code example

Python code showing the use of recursion to calculate factorial

```python
def factorial(n):
    # This is the base case of the method
    if n == 0:
        return 1

    # Here the recursion happens
    return (n * factorial(n -= 1))
```
